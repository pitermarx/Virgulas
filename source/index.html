Keyboard Shortcuts

  • EnterAdd sibling / first child
  • Enter (no focus)Add root node
  • BackspaceDelete empty node
  • Ctrl+BackspaceDelete node (confirm if it has children)
  • TabIndent
  • Shift+TabOutdent
  • ↑ / ↓Navigate
  • Shift+↑/↓Multi-select siblings
  • DeleteDelete selected nodes
  • Alt+↑/↓Move node
  • Alt+→Zoom in
  • Alt+←Zoom out
  • EscClear focus / toggle search
  • Ctrl+SpaceCollapse / expand
  • Shift+EnterEdit description
  • Search: Tab / Shift+Tab or ↑ / ↓Cycle results
  • Search: EnterOpen selected result
